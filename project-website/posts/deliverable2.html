Personas

Professor Kohee Sang

  • 35-year-old professor of Computer Science, male
  • PhD in Computer Science from University of Toronto
  • Teaching stream, responsible for making and grading assignments
  • Comfortable with technology, prefers it to pen and paper
  • Owns multiple Macbooks, works on laptops primarily
  • Wants to create assignments and view results
  • Wants to view statistics about class performance and engagement
  • Mostly teaches courses related to computers, math and technology
  • Single, enjoys tinkering with software in his spare time

Professor Shierry Tans

  • 45-year-old professor of Statistics, female
  • PhD in Statistics from York University
  • Research stream, teaching is not her number one priority
  • Prefers old-fashioned way of handling assignments, doesn't like interacting with software
  • Usually works with pen and paper, organized through notebooks and calendars
  • Wants to create assignments and view results
  • Just wants to view basic grades and class averages for individual assignments
  • Teaches math and stats
  • Has a family of 4, doesn't have a lot of time to tinker with software

John Doe, Undergraduate

  • 19-year-old undergraduate student
  • Studying for a Psychology degree
  • This course is a program requirement
  • Only use social media for necessity like face to face chat more
  • Does not have kids but is married
  • Lives off campus that's far away

Nick Harrington, B.Sc. Candidate

  • 20-year old Statistics student, male
  • Currently in 2nd year for a degree in Statistics, Machine Learning Stream from the University of Toronto
  • Canadian, speaks fluent English and conversational French
  • Maintains a 4.0 cGPA, looking to attend graduate school
  • Obsessed with academic performance through constant self-testing and grade tracking charts on his computer and phone
  • Uses offline Android phone during commute to study through flashcards
  • Does not like flashcards that much since statistics is mostly not about memorization
  • Very comfortable with technology, prefers customizability to modify programs to his liking
  • Prefers having power over his apps over a pretty interface
  • However, strongly dislikes if the interface is borderline unusable (Blackboard)
  • Sometimes prone to looking up answers on WolframAlpha when stuck on a question

User Stories

  • As a User, I want to be able to sign up for an Account using a utorid and password, and log into and out of my account.
  • As a User I want to be able to see my privilege and access rights
  • As Professor Shierry Tans, I want to save statistics questions so that I can used them later or next year.
  • As a Professor Kohee Sang I want to be able to create a class which generates a unique code that I can give to my students so that they can join the class and do their work.
  • As Professor Kohee, I want to be able to manage my online courses from multiple computers
  • As John, a student; I want to be able to join a professor's course through a unique code so that I can participate in the online assignments
  • As Nick, a student; I want to see a listing of all my online courses so that I can be more organized
  • As Professor Shierry Tans, I want to easily create different type of question to test my students. I.e multiple choice, fill in the blank, graphing and paragraphs
  • As Professor Kohee Sang, I want to be able to create an assignment with a due date, rubric, and set of questions. to assign to a course
  • As a Professor Shierry Tans, I want to indicate which chapter of the textbook or lecture slides is relevant for the question
  • As a Professor Kohee Sang, I want to submit question to the bank using Latex script as I am familiar with it
  • As Professor Shierry Tans, I want to be able to upload images of graphs/photos so that the questions can be more dynamic
  • As Professor Kohee Sang, I want to be able to create an answer key so that questions and assignments can be auto-marked.
  • As John, a student; I want to be able to submit my completed assignments for marking.
  • As Nick, a student; I want to be able to attempt assignments an infinite amount of times so that I can get a lot of practice answering questions
  • As John, a student; I want to be able to refresh an assignment to get a new set of questions.
  • As Nick, a student; I want to be able to view my marks for my previous assignments so that I can calculate my average.
  • As John, a student; I want to be able to save an assignment for completion at a later date.
  • As Professor Kohee Sang I want to be able to provide notices of hints/feedback for certain type of questions when the student did the question wrong.
  • As Professor Kohee Sang, I want to able to see and manage student's marks
  • As Professor Shierry Tans, I want to see the completion rate and correction rate of all questions and class averages at a glance of an assignment
  • As Nick, a student; I want to be able to add certain questions as favourites so I can come back to it later for practices.
  • As John Doe, a student; I want to be able to earn badges for completing assignments to make doing them more fun and engaging.
  • As John, a student; I want to be able to search and insert latex math scripts for ease of use.
  • As Nick, a student; I want to be able to download a copy of the problem set, so that I am able to print out the problem and practice them where ever I want in hard copy.
  • As Nick, a student; I want to be able to email the Professor under certain problem if there are question I want to ask or if certain error occurs on the website.
